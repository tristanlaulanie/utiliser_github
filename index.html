Exemples d'utilisation de GIT





Création d'un projet GIT depuis le poste de travail.



  1. Se placer dans le dossier de travail
  2. git init
  3. Créer un README.md (pour avoir du contenu)
  4. Faire un commit (commit initial)
  5. Créer un dépôt git sur GitHub
    On repasse sur le terminal.
  6. git remote add origin URL_DU_DEPOT_DISTANT (pour relier les deux)
  7. git branch -M main
  8. git push -u origin main
Désormais, votre dépôt git local, est relié à votre dépôt git distant


Création d’un projet GIT depuis GitHub

  1. Aller sur GitHub.com
  2. Créer un nouveau dépôt, en cochant la case pour avoir un README.md
  3. Aller sur la page d’accueil du projet
  4. Récupérer le lien du dépôt en cliquant sur le bouton
  5. Se placer dans un dossier qui contiendra le dossier de travail
  6. git clone URL_DU_DEPOT_DISTANT
Le dossier du projet a été créé, il contient la version actuelle du dépôt GitHub (c’est un clone).

Faire un commit

  1. git add.(ajoute tous les fichiers du dossiers contenant des modifications au prochain commit)
  2. git commit -m « message » (créé une sauvegarde comprenant le contenu des anciens commits + les changement ajoutés avec la commande add).

Créer une branche

On peut créer une branche en utilisant les commandes :

- git branch NOM_DE_LA_BRANCHE
- git checkout -b NOM_DE_LA_BRANCHE

Vous pouvez changer de branche en utilisant git checkout NOM_DE_LA_BRANCHE

Attention, on ne peut pas créer de branche dans un dépôt GIT vide (CAD qui ne contient aucun commit).

Récupérer | Envoyer des données avec le dépôt distant

On peut mettre à jour le dépôt local par rapport au dépôt distant avec la commande :
git pull

On peut mettre à jour le dépôt distant par rapport au dépôt avec la commande :

git push

Ces commandes synchronisent les commit si des fichiers locaux ne font partie d’aucun commit, ils ne seront pas envoyés dans le dépôt distant.
